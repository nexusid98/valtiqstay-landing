# ValtiqStay - Roadmap MVP Trentino

## Overview
App di check-in veloce con QR code per hotel trentini + sito web per recensioni e loyalty points. Go-to-market in 8-10 settimane.

**Constraint:** 1 founder, 20 ore/settimana, sviluppo con Claude, MVP pragmatico.

---

## FASE 1 - SETTIMANE 1-2: Research & Design
**Obiettivo:** Architettura e design definiti

- Definire tech stack (React Native o Flutter per app)
- Wireframe check-in QR + form ospiti
- Schema database: ospiti, hotel, check-in, punti loyalty
- API design

**Ore settimanali:** 4-5  
**Output:** Documento tecnico + mockup  
**Owner:** Tu + Claude

---

## FASE 2 - SETTIMANE 3-5: App Mobile MVP
**Obiettivo:** App funzionante con QR code e capture dati ospiti

- QR code reader integrato
- Form check-in minimalista (nome, email, data arrivo/partenza)
- Storage dati locale
- UI semplice ma polished

**Ore settimanali:** 6-7 (review + testing + feedback loops)  
**Output:** App testabile su device  
**Owner:** Claude (dev), Tu (QA + validazione)

---

## FASE 3 - SETTIMANE 6-7: Backend + Database
**Obiettivo:** Dati sincronizzati tra app e cloud

- Setup Supabase o Firebase (gratis tier)
- API per ricevere dati check-in
- Dashboard hotel minimalista (vedere chi ha fatto check-in)
- Autenticazione hotel basic

**Ore settimanali:** 5-6  
**Output:** App sincronizza con backend  
**Owner:** Claude (dev), Tu (testing)

---

## FASE 4 - SETTIMANE 8-10: Sito Web + Launch Prep
**Obiettivo:** Platform pronta per primo hotel pilota

- Sito signup hotel (landing semplice)
- Dashboard hotel basic (view check-in, dati ospiti, punti)
- Pagina ospiti per vedere punti (minimalista)
- Piano commerciale primo hotel pilota

**Ore settimanali:** 4-5  
**Output:** Sistema completo pronto per beta  
**Owner:** Claude (dev), Tu (product)

---

## SETTIMANA 11: Beta Launch
Primo hotel trentino pilota live con ValtiqStay.

---

## KPI MVP
- Check-in processati via QR
- Dati ospiti raccolti correttamente
- Hotel vede dashboard senza frizione
- <2 min per check-in

## Note
- No pagamenti integrati in MVP (solo contanti in hotel)
- No loyalty rewards partner in MVP (solo punti base)
- No sito web pubblico in MVP (solo accesso hotel/ospiti invitati)
- Tutto perfettibile in fase di crescita post-launch
